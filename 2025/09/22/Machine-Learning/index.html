Machine Learning Overview

Supervised Learning

x -> y Learn from data labeled with the “right answers”. 通过提供正确的答案，算法学习给定的输入 X 和正确标签 Y，最终学会在只有输入时做出准确的预测输出

两个主要类别 Regresion(回归)，Classification(分类)

Unsupervised learning

Find something interesting in unlabeled data. Data only comes with inputs x, but not output labels y. Algorithm has to find structure in the data. 三个主要类别， Clustering(Group simliar data points together), Anomaly detection(Find unusual data points), Dimensionality reductuon(Compress data using fewer numbers)

Linear Regreesion Model

Terminology

Training set : Data used to train the model.

Notation: x = “input” veriable feature

y = “output”/“target” variable

m = number of training examples

(x, y) = single training example

(x(i), y(i)) = ith training example

xj = jthfeature

n = number of features

x⃗(i) = features of ith training example

Linear Regreesion with one input

x -> f -> $ $

fw,b(x) = wx + b

Cost Function : Squared error cost function

Find w,b: $ $(i) is close to y(i) for all (x(i), y(i))

$$ J(w,b) = \frac{1}{2m} \sum_{i=1}^{m} \left( \hat{y}^{(i)} - y^{(i)} \right)^2 $$

Gradient Decent

$$ w = w - \alpha \frac{d}{dw} J(w,b) $$

$$ b = b - \alpha \frac{d}{db } J(w,b) $$

Linear Regreesion with Multiple Variables

fw⃗, b(X⃗) = w⃗ ⋅ X⃗ + b

$$ w_j = w_j - \alpha \frac{d}{dw_j} J(\vec{w},b) $$

$$ b = b - \alpha \frac{d}{db } J(\vec{w},b) $$

Normal equation:

  • only for linear regression
  • solve for w, b without iteration
