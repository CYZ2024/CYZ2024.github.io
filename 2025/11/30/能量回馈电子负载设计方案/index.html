能量回馈电子负载设计方案

发表于 2025-11-30 分类于 学习笔记

A题：能量回馈的变流器负载试验装置设计方案报告

1. 方案摘要

本方案针对”能量回馈的变流器负载试验装置”赛题要求，设计了一套基于背靠背双PWM变流器（Back-to-Back Converters）架构的高效能测试系统。

系统采用三相全桥逆变器作为变流器1，三相电压型PWM整流器（VSR）作为变流器2。主控单元选用32位浮点DSP（或高性能混合信号MCU），利用SVPWM（空间矢量脉宽调制）技术和DQ旋转坐标系下的双闭环解耦控制，在实现高精度三相输出（THD < 1%）的同时，通过有源整流实现单位功率因数能量回馈。该设计最大化降低了直流电源的输出功率，实现了高效节能与高精度测试的完美结合。

2. 系统总体设计

2.1 系统架构

系统主要由直流稳压电源、变流器1（DC-AC三相逆变）、三相LC滤波器、连接单元（模拟传输线/电感）、变流器2（AC-DC三相有源整流）及反馈回路组成。

  • 前级（变流器1）： 运行于恒压恒频（CVCF）模式。负责将直流电源逆变为满足指标要求的三相交流电（线电压32V, 频率50Hz），建立稳定的交流微网电压，作为被测源。
  • 后级（变流器2）： 运行于恒流（CC）模式。作为电子负载，负责从交流侧汲取正弦电流，并将其整流回直流，以电流源形式向直流母线回馈能量。

2.2 核心技术特征

  1. 全控型背靠背拓扑： 变流器1与变流器2均采用三相全桥结构（全控型MOSFET），摒弃了传统的二极管整流方案。这使得系统不仅能双向流动能量，还能精确控制电流波形，从根本上保证了交流侧电流的正弦度。
  2. SVPWM 调制技术： 相比传统SPWM，SVPWM算法不仅提高了直流电压利用率（约15%），还显著降低了开关损耗和谐波含量，优化了输出电能质量。
  3. DQ轴解耦控制： 在同步旋转坐标系下将三相交流量转换为直流分量进行PI调节，实现了有功电流和无功电流的独立控制，消除了稳态误差，大幅提升了系统的动态响应速度和稳压/稳流精度。

3. 理论分析与参数计算

3.1 变流器1（三相逆变器）设计

设计目标： 输出线电压有效值 Uline_rms = 32V，频率 f = 50Hz。

  1. 直流母线电压设定：
    采用SVPWM调制，线性调制区内，直流母线电压 Udc 需满足：

    考虑死区效应、线路压降及调制裕量，推荐直流电源电压设定在 50V-60V。

  2. LC滤波器参数：
    为滤除开关频率的高频谐波，截止频率 fc 应满足 10fout ≤ fc ≤ fsw/10。
    设定开关频率 fsw = 20kHz，选取截止频率 fc ≈ 1.5kHz。

    选型结果： 滤波电感 L = 1mH，滤波电容 C = 10μF（Y型接法）。

3.2 变流器2（能量回馈整流器）设计

设计目标： 模拟负载电流 I1（最大2A），并将能量回馈至直流侧。

  • 控制策略： 采用电压定向矢量控制（VOC）。变流器2通过锁相环（PLL）实时检测变流器1输出电压的相位，控制自身输入电流 I1 的相位与电压相位严格同步（模拟纯电阻负载）。
  • 回馈机制： 变流器2控制交流侧电流流向直流侧，使得直流母线电容电压有升高趋势，从而自然向并联的直流稳压电源反向注入电流，实现能量循环。此时直流电源仅需补充两级变流器及线路的损耗功率。

4. 硬件电路设计

4.1 主功率回路

系统包含两套完全对称的三相全桥电路，共使用12个功率MOSFET。

  • 功率器件（MOSFET）： 选用高耐压、低导通电阻的N沟道MOSFET，例如 HYG025N10NS1P (100V, 2.5mΩ) 或 IPP023N10N5。
    • 选型理由： 100V耐压提供了充足的安全裕量，极低的 RDS(on) 能有效减少导通损耗，提升系统整体效率。
  • 驱动电路： 采用带死区控制的高性能半桥驱动芯片，如 IR2104 或 EG2133。每相独立驱动，配置低ESR的自举电容（1μF 钽电容）以确保高频开关下的驱动稳定性。

4.2 采样与调理电路

高精度的信号采集是实现闭环控制的基础。

  • 交流电压采样： 采用高精度线性光耦或差分运放电路，实时采集三相线电压或相电压。
  • 交流电流采样： 选用高带宽霍尔电流传感器（如 ACS712-05B 或 CH701），串联在滤波器电感前端，确保电流波形还原度。
  • 信号调理： 包含二阶有源低通滤波及电平抬升电路，滤除高频噪声并将双极性交流信号转换为MCU ADC可接受的单极性信号（0-3.3V）。

4.3 主控模块

  • 核心控制器： 选用 STM32G474 系列或 TI TMS320F28335 DSP。
    • 选型理由： 该类芯片专为数字电源控制设计，内置高分辨率定时器（HRTIM）、硬件三角函数加速器（CORDIC）及高速ADC，能够轻松应对双变流器的高频SVPWM运算与多路同步采样需求。

5. 软件控制策略

5.1 变流器1控制逻辑（逆变源）

  1. 坐标变换： 采集三相电流 Iabc 和电压 Uabc，执行 Clark (abc → αβ) 和 Park (αβ → dq) 变换。
  2. 双闭环控制：
    • 外环（电压环）： Ud 给定值设为 ，Uq 给定为0。PI控制器输出 d 轴参考电流。
    • 内环（电流环）： 将参考电流与实际反馈电流进行PI调节，输出电压矢量指令。
  3. SVPWM生成： 根据电压矢量指令，利用空间矢量算法计算六路PWM占空比，驱动全桥电路。

5.2 变流器2控制逻辑（有源负载/回馈）

  1. 软件锁相环（SPLL）： 基于SOGI（二阶广义积分器）或dq锁相环，精准锁定变流器1输出电压的相位角 θ。
  2. 电流指令给定：
    • Iq* = 0：确保无功分量为0，实现单位功率因数运行（纯阻性）。
    • Id*：根据测试需求设定（如2A），控制有功电流大小。
  3. 能量回馈控制： 通过双闭环控制使得变流器2从交流侧吸收功率，经过全桥整流提升直流侧电压，实现能量自动回馈至直流源。

6. 测试与调试步骤

  1. 变流器1独立调试：
    • 断开变流器2，仅对变流器1上电。
    • 验证SVPWM波形及死区时间（1us-2us）。
    • 带阻性负载测试，调节电压环PI参数，确保输出 32V ± 0.25V，频率 50Hz，THD < 2%。
    • 测试0A-2A负载突变下的动态响应，确保负载调整率 SI1 ≤ 0.3%。
  2. 系统联调（能量回馈）：
    • 接入变流器2，直流端并联，交流端通过连接单元连接。
    • 逐步增加变流器2的电流给定值 Id*。
    • 监测关键指标： 观察直流电源输出电流 Isource。在理想回馈状态下，当交流电流 I1 = 2A 时，直流电源仅需提供系统损耗功率（预计10W-20W），显著低于无回馈时的总功率。
    • 微调电流环带宽，消除电流波形尖峰，确保回馈过程平稳。

7. 结论

本方案采用了先进的三相全桥背靠背拓扑与矢量控制技术，构建了一套高性能的能量回馈变流器负载试验装置。该设计不仅实现了高精度的三相交流稳压输出，更关键的是通过有源整流技术实现了高质量的能量回馈，将传统的”耗能型”测试转变为”节能型”测试。系统各项指标均满足并优于赛题要求，具有极高的工程应用价值。
